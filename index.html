Google Spreadsheet Sync Setup

To capture live portfolio data directly inside your exact target spreadsheet (**ID: 1LS26OhgKRv0wz_WxDjessqRB_cFfXfPMB_S1zFHWZ-0**), follow these simple directives:

  1. Go to Google Sheets and open your target spreadsheet:
    Open Spreadsheet Link
  2. In the menu, click on Extensions > Apps Script.
  3. Clear any existing code and paste the exact code script shown below:
  4. Click Deploy > New Deployment. Choose Web App, change Access to "Anyone", and click Deploy.
  5. Copy the generated Web App URL and paste it inside the googleScriptApiUrl config parameter at the bottom of this file!
function doPost(e) {
  var sheetId = "1LS26OhgKRv0wz_WxDjessqRB_cFfXfPMB_S1zFHWZ-0";
  var sheet = SpreadsheetApp.openById(sheetId).getActiveSheet();
  var d = JSON.parse(e.postData.contents);
  sheet.appendRow([new Date(), d.name, d.phone, d.email, d.message]);
  return ContentService.createTextOutput(JSON.stringify({"result":"success"}))
    .setMimeType(ContentService.MimeType.JSON);
}
